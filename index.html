It is supposed that it is PARAXIAL APPROXIMATION.

In the curved interface matrix, if the radius of curvature(R) is infinite, please enter the 'w'.

If you input the matrices M1, M2, M3.... and Mn in order, it will be calculated as M_sys = Mn*...*M2*M1.

If you input the matrices such as M_sys = M_image*M(arbitrary matrix)*M_object and get B = 0, you can get the Imaging Magnification(A).

For accurate calculation, please express as fractions.
